```bash
npm init vue@latest
```

Add `@rugo-vn/vue` package.

```bash
cd your-project
npm uninstall -S vue vue-router
npm i
npm i -D @rugo-vn/vue
```

## Example

Create your own admin page: Visit `example` directory in our source for more information.

```bash
cd example/
npm i
npm run dev
```

## Development

For building `@rugo-vn/vue` only.

```bash
npm run build:css
npm run build:ionicons
```
