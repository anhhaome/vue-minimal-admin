# Rugo Vue

@rugo-vn/vue

## Generate Library

```bash
npm run clean
npm run build
```

## Generate Documents

_Required: Generate Library_

```bash
npm run doc:generate
```

## Install Example

```bash
npm run example:clean
npm run example:install
```

Run example:

```bash
npm run example
```

## Publish to NPM

_Required: Generate Library_

Increase version number.

```bash
npm run publish:beta
```

Or 

```bash
npm run publish:release
```
